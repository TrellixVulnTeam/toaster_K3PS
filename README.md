# Toaster - Simple macOS Package Manager

## Installation

## macOS

```bash
chmod +x ./installer.zsh && sh ./installer.sh
```

## Reinstall

## macOS

```bash
chmod +x ./uninstaller.sh && chmod +x ./installer.sh && sh ./uninstaller.sh && sh ./installer.sh
```
